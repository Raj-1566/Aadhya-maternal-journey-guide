# Overview

Aadhya is a web-based support platform designed to empower mothers through community connection, emotional well-being, and practical guidance. The frontend serves as the user interface for mothers to access support resources, connect with experts, and interact with a nurturing digital environment.

This repository contains the frontend implementation of the Aadhya platform, built with simplicity, empathy, and accessibility in mind.

# Objectives

Create an intuitive and comforting user interface for mothers.

Enable easy access to forums, articles, and mental health support.

Connect mothers to professionals, resources, and peer groups.

Promote community engagement and emotional support.
